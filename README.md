# ZZP 2018

This is the student project. It is created for training cooperation in project team, creating game and learning SDL2.

## Downloading

```bash
git clone https://github.com/pantadeusz/zzp2018a
```

## Compilation

```bash
mkdir build
cd build
cmake ..
make
```

## Running

```bash
./zzpsopotapp
```
